# 🚀 MindQuest Hub – Dynamic Quiz App

A modern, interactive quiz platform built using **HTML, CSS, and JavaScript**, designed to deliver a smooth user experience with dynamic data and real-time feedback.

---

## 🔗 Live Demo

👉 [View Live Project](https://quizzzi.netlify.app/)
 

---

## 🔥 Features

* 🎯 **Dynamic Quiz App**
  Questions are fetched in real-time using an external API.

* 🎨 **Clean UI + UX**
  Minimal, responsive, and user-friendly interface.

* 🌐 **API Integration**
  Uses Open Trivia DB API to load fresh questions dynamically.

* 🧠 **State Management**
  Tracks current question, score, and user responses efficiently.

* 🌙 **Theme Toggle**
  Switch between light and dark mode for better usability.

* ⏱️ **Timer + Progress Graph**
  Countdown timer for each quiz and visual performance feedback.

* ⚙️ **Real-world Features**
  Includes navigation, result analytics, and answer review system.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Open Trivia API

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Mannu-Thakur/Quiz_Portal.git
```

### 2. Navigate to project

```bash
cd Quiz_Portal
```

### 3. Run the app

Open `index.html` in your browser.

## 💡 How It Works

* Tracks user answers and score

* Provides answer review for learning
